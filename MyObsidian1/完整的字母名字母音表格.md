## 二、哪些字母最容易混淆

### 1. 字母名和字母音完全不同

|字母|字母名|字母音|
|---|---|---|
|A|/eɪ/|/æ/|
|E|/iː/|/e/|
|I|/aɪ/|/ɪ/|
|O|/oʊ/|/ɒ/|
|U|/juː/|/ʌ/|

这5个字母就是：

**A E I O U**

也就是英语中的 **元音字母（Vowels）**。

---

### 2. 字母名里就带着字母音

例如：

|字母|字母名|字母音|
|---|---|---|
|B|/biː/|/b/|
|D|/diː/|/d/|
|P|/piː/|/p/|
|T|/tiː/|/t/|

规律：

> 字母名 = 字母音 + /iː/

所以孩子学起来比较容易。

## 三、自然拼读最先学的 42 个核心音

学完26个字母音后，下一阶段不是背单词，而是学习：

### 单字母音

```
a  /æ/
e  /e/
i  /ɪ/
o  /ɒ/
u  /ʌ/

b  /b/
c  /k/
d  /d/
...
```

### 双字母组合音

```
sh  /ʃ/
ch  /tʃ/
th  /θ/
/ð/
ng  /ŋ/
ck  /k/
```

### 元音组合

```
ai  /eɪ/
ee  /iː/
oa  /oʊ/
oo  /uː/
ar  /ɑː/
or  /ɔː/
```

这些组合音掌握后，大部分陌生单词都能拼出来。

## 四、学习顺序（推荐）

不要按 ABCDEFG 学。

推荐按自然拼读体系学习：

第一组：

```
s
a
t
i
p
n
```

能直接拼读：

```
sat
sit
sap
sip
pin
pan
tap
tip
```

第二组：

```
c
k
e
h
r
m
d
```

第三组：

```
g
o
u
l
f
b
```

第四组：

```
j
v
w
x
y
z
qu
```

这样学到第2周左右，就已经可以自己拼读几十个单词了，而不是只会背26个字母。

你现在已经理解了整体框架，下一步最适合的是学习：

**26个字母音的发音器官位置（口型、舌位、气流）+ 最常见例词。**

这样以后看到单词，就能从“字母 → 发音 → 拼读 → 单词”建立完整链路。
