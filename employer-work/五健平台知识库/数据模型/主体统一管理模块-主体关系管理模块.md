# 模块 1：Entity Registry（主体统一管理）

这是 **主数据中心（Master Data）**，也是全系统的根。

---

## 最终表清单（生产级）

```text
entity
entity_type
entity_identifier
entity_extend_field
entity_audit_log
```

```text
共：5 张表
```

---

## 1. entity（主体主表）

```sql
CREATE TABLE entity (

    id BIGINT PRIMARY KEY,

    entity_code VARCHAR(64) UNIQUE,

    entity_name VARCHAR(255),

    entity_type_id BIGINT NOT NULL,

    status SMALLINT NOT NULL,

    gender VARCHAR(16),

    birthday DATE,

    created_at TIMESTAMP NOT NULL,

    updated_at TIMESTAMP,

    deleted_at TIMESTAMP
);
```

---

## 2. entity_type（主体类型）

```sql
CREATE TABLE entity_type (

    id BIGINT PRIMARY KEY,

    type_code VARCHAR(64) UNIQUE,

    type_name VARCHAR(128),

    description TEXT,

    is_active BOOLEAN,

    sort_order INT,

    created_at TIMESTAMP
);
```

---

## 3. entity_identifier（主体标识）

用于解决：

```text
一个主体多个ID
```

例如：

```text
student_id
id_card
phone
wechat_openid
hospital_no
```

```sql
CREATE TABLE entity_identifier (

    id BIGINT PRIMARY KEY,

    entity_id BIGINT NOT NULL,

    identifier_type VARCHAR(64) NOT NULL,

    identifier_value VARCHAR(128) NOT NULL,

    is_primary BOOLEAN DEFAULT FALSE,

    created_at TIMESTAMP
);
```

---

## 4. entity_extend_field（动态属性）

这是：

```text
系统可扩展性的核心
```

例如：

```text
height
weight
BMI
school
grade
class
```

```sql
CREATE TABLE entity_extend_field (

    id BIGINT PRIMARY KEY,

    entity_id BIGINT NOT NULL,

    field_code VARCHAR(64),

    field_value TEXT,

    created_at TIMESTAMP
);
```

---

## 5. entity_audit_log（主体审计日志）

```sql
CREATE TABLE entity_audit_log (

    id BIGINT PRIMARY KEY,

    entity_id BIGINT NOT NULL,

    operation VARCHAR(64),

    operator_id BIGINT,

    before_data JSON,

    after_data JSON,

    created_at TIMESTAMP
);
```
