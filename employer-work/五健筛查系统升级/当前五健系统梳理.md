||||







|参考范围类型|适用结果类型|规则表达方式|示例指标|示例配置|
|---|---|---|---|---|
|单阈值（threshold）|numeric / score / count|value ≥ / ≤ X|龋齿数量（CARIES_COUNT）|≥1 为异常|
|区间范围（range）|numeric / decimal|min ≤ value ≤ max|裸眼视力（VISION）|4.6–5.0 正常|
|多区间等级（range_level）|numeric / score|多区间映射等级|BMI|<P5 / P5–P85 / ≥P95|
|百分位（percentile）|decimal|P值比较|BMI（儿童）|≥P95 肥胖|
|枚举映射（enum_map）|enum|枚举值→结果|脊柱侧弯|异常=阳性|
|布尔判断（boolean）|boolean|true / false|是否戴眼镜|true=风险|
|组合规则（composite）|多字段|AND / OR 条件|近视判定|视力<5.0 且戴镜|
|多值包含（multi_select）|array|包含某值|口腔问题|包含“龋齿”|
|文本匹配（text_match）|text|关键词匹配|医生备注|包含“异常”|



|结果类型|数据类型|示例指标|示例值|
|---|---|---|---|
|数值型（numeric）|number|身高（HEIGHT）|140|
|数值型（decimal）|decimal|BMI|20.4|
|区间型（range）|decimal|裸眼视力（VISION_RIGHT）|4.8|
|布尔型（boolean）|boolean|是否佩戴眼镜（WEAR_GLASSES）|true|
|枚举型（enum）|string|脊柱侧弯（SCOLIOSIS_RESULT）|正常 / 异常|
|等级型（level）|string|营养状况（NUTRITION_STATUS）|正常 / 超重|
|计数型（count）|integer|龋齿数量（CARIES_COUNT）|2|
|评分型（score）|integer|心理总评分（PSY_TOTAL_SCORE）|65|
|多选型（multi-select）|array|口腔问题类型|龋齿、牙龈出血|
|文本型（text）|string|医生备注|轻度异常，建议复查|
