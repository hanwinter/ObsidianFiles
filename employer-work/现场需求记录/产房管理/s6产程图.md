产程图的价值与意义： 通过绘制产程图，动态监控产妇分娩过程中宫口扩张、先露高低、胎心等信息，达到实时判断产程中的风险的目的； 产后归档记录
现在产科实际的操作：在产妇生产过程中只是记录信息，产后补录产程图，没有达到实时监测的目的。
应用孕产保健系统后产程图的设计与价值：应该发挥最初的价值，在录入产程数据后，在同一个页面中实时更新产程图，实现动态检测的目的； 产后按上级要求打印出归档用的产程图

```

**优化后表述**
产程图是产科分娩管理过程中重要的临床监测工具，通过对宫口扩张、胎先露下降、胎心、宫缩等关键指标进行连续记录和图形化展示，直观反映产程进展情况，为医护人员判断产程是否正常、识别产程异常及胎儿风险提供辅助依据，同时作为分娩病历的重要组成部分进行归档保存。

在传统纸质管理模式下，医护人员主要记录产程观察数据，产程图的绘制和整理工作存在一定滞后性，产程图更多承担病历记录和归档功能，其动态监测价值未得到充分发挥。

在孕产保健管理系统中，医护人员录入产程观察数据后，系统自动实时生成电子产程图，并在同一工作界面动态展示产程变化趋势；同时结合产程监测规则，对宫口扩张停滞、胎心异常、第二产程延长等情况进行预警提示，辅助医护人员及时发现和处理产程风险。分娩结束后，系统自动生成标准产程图，可打印归档或纳入电子病历保存，实现“实时监测、智能预警、自动归档”的一体化管理模式。
```

分阶段：宫口扩张 (cm) + 胎头 S 值（先露高低）完整范围

说明：S 是胎头位置，**S 负 = 坐骨棘上方，S0 = 平坐骨棘，S 正 = 坐骨棘下方**，无 “胎头下降厘米” 这种口语表述，统一产科标准 S 值。

## 1. 待产（未正式临产，有不规则宫缩 / 先兆临产）

- 宫口扩张：0～1cm（大多闭合 0cm，经产妇可松弛至 1cm，宫颈管未完全消退）
- 胎头 S 值范围：S-3 ～ S-1（胎头高浮，未入盆 / 浅入盆）

## 2. 第一产程（规律宫缩至宫口全开 10cm，分潜伏期 + 活跃期）

1. 潜伏期（0～3cm）
	- 宫口：0～3cm
	- 胎头：S-2 ～ S0

2. 活跃期（3～10cm）
	- 宫口：3～10cm
	- 胎头：S0 ～ S+2

**第一产程整体汇总范围**
- 宫口扩张：0 ～ 10cm
- 胎头 S 值：S-2 ～ S+2

## 3. 第二产程（宫口开全 10cm → 胎儿娩出）

- 宫口扩张：固定 10cm（全开不再变化）
- 胎头 S 值范围：S+2 ～ S+4（S+3、S+4 接近阴道口，胎儿即将娩出）

## 4. 第三产程（胎儿娩出 → 胎盘胎膜完整娩出，5～15 分钟）

- 宫口扩张：维持 10cm，胎盘娩出后逐步回缩松弛
- 胎头：胎儿已经娩出，无胎头位置 S 值，该指标无意义、不记录

## 5. 第四产程（胎盘娩出后产后 2 小时产房观察期）

- 宫口：宫颈开始回缩，宫口由 10cm 缓慢回缩至 2～3cm，后续持续闭合
- 胎头：胎儿已娩出，无胎头下降 S 值，不采集该指标

## 补充业务规则（系统判断用）

1. 产程图仅绘制**第一、第二产程**的宫口曲线、胎头下降 S 曲线；
2. 第三、第四产程界面不再展示胎头 S 值输入框；
3. 若第一产程宫口扩张停滞、胎头 S 值长时间无下降，系统自动标记产程异常，纳入高危预警。
